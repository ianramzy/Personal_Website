Instructions

  1. Clone project on github

  2. If you don't already have BlueJ installed (which you probably don't) now would be a good time to install it, you'll want version 3.1.7.

  3. Now open the project with BlueJ and right-click run it as an applet. Happy rendering! (known bug: sometimes the window will only update the rendered curve when you resize it)

Inspiration

I wanted to make a fractal generator and what better fractal to use than the famous Koch Curve, mathematically one of the first fractals ever discovered!

What it does

It renders a Koch Curve using recursion to a user specified number of iterations.

How it's built

I created this in a BlueJ Applet (Java).

Future ideas and what's next

There are a bunch of way cooler fractals to render, I would like to implement some of those next.
