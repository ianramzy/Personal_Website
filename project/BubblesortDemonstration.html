Instructions

  1. Clone project on github

  2. If you don't already have BlueJ installed (which you probably don't) now would be a good time to install it, you'll want version 3.1.7.

  3. Now open the project with BlueJ and right-click run it as an applet. Happy sorting!

Inspiration

When I was first learning about the various sorting algorithms I would look online to see videos of how they would sort data, so I decided to try and make one myself.

What it does

It is a visualization of the bubblesort algorithm, the up and down keys will change the delay of the algorithm (making it go either faster or slower). The current bar being sorted is highlighted green, and once it is sorted into the list it is highlighted red. If you're not familiar with sorting algorithms have a look at how bubblesort works here.

How it's built

I created this in BlueJ which is a version of Java, the program itself is a Java Applet.

Future ideas and what's next

In the future I would like to implement different algorithms to visualize such as quicksort.
