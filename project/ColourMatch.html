Instructions to Play

  1. Click the panel below to activate flash and click allow flash on the webpage. You may need to add this site to the list of sites allowed to use flash player in your browsers settings.

  2. To play click or use the spacebar to rotate the cube to match the colours! Try and last as long as possible against the falling balls!

Instructions to Edit and view source

  1. Clone project on github

  2. Open and view in Adobe Animate and setup project as ActionScript 3.0

Inspiration

I saw the games that were dominating the App store and Play store at the time and I wanted to try and emulate the style of simple yet challenging game play.

How it's built

I built this in one night using only ActionScript 3.0 (flash).

Future ideas and what's next

In the future I would like to add options for different colour modes, as well as maybe different shapes. Implementing a level system could also be interesting.
